# AI RPG Game 🎲

这是一个基于大语言模型（LLM）的单文件网页版角色扮演游戏引擎。通过 AI 强大的叙事能力，结合经典的 RPG 数值体系（属性、技能、物品、战斗），为你带来无限可能的冒险体验。

## ✨ 主要特性

*   **多模型支持**：无缝支持 Google Gemini、DeepSeek、Anthropic (Claude) 和 OpenAI (GPT) 等主流 AI 模型。
*   **智能角色生成**：只需输入世界观（如“赛博朋克”、“高魔奇幻”）和角色描述，AI 将自动生成包含属性、技能和背景故事的完整角色卡。
*   **混合游戏系统**：
    *   **D20 核心规则**：内置骰子模拟器，支持属性检定和技能成长。
    *   **双模式战斗**：支持纯 AI 叙事战斗，或启用 JavaScript 确定性伤害计算，满足不同玩家对随机性和策略性的需求。
*   **完整 RPG 体验**：
    *   动态背包与物品系统（支持消耗品使用）。
    *   技能系统（主动/被动、Buff/Debuff 状态管理）。
    *   探索与战斗阶段自动切换。
*   **存档管理**：支持导出/导入 JSON 格式的游戏进度，随时保存你的史诗冒险。
*   **现代化 UI**：基于 React 和 Tailwind CSS 构建的响应式暗黑风格界面。

## 🚀 快速开始

本项目为**单文件**应用，无需安装 Node.js 或配置复杂的后端环境。

1.  下载本项目中的 `index.html` 文件。
2.  直接使用现代浏览器（Chrome, Edge, Firefox 等）打开该文件。
3.  在启动界面配置你的 AI API Key 即可开始游戏。

## ⚙️ 配置说明

在游戏启动页，你需要选择 AI 提供商并填入对应的 API Key：

*   **Google (Gemini)**: 推荐使用 `Gemini 1.5 Pro` 或 `Gemini 2.5 Pro`。需申请 Google AI Studio Key。
*   **DeepSeek**: 支持 DeepSeek V3/R1。需填写 DeepSeek API Key。
*   **Claude (Anthropic)**: *注意：由于 CORS 限制，浏览器直接调用可能受限，建议配置反向代理 Base URL。*
*   **OpenAI / Custom**: 支持标准 OpenAI 格式的 API，可用于连接本地模型（如 Ollama, LM Studio）或其他兼容服务。

## 🎮 游戏玩法指南

1.  **角色创建**：输入你想体验的世界背景（例如“废土世界”）和角色设定（例如“寻找水源的变异游侠”），点击生成。
2.  **探索模式**：在输入框描述你的行动（如“向北走”，“调查神庙”）。AI 会根据 D20 规则判断成功率并推进剧情。
3.  **战斗模式**：
    *   当遭遇敌人时自动进入战斗。
    *   使用**技能栏**中的技能进行攻击或辅助。
    *   勾选 **JS伤害** 可使用传统游戏逻辑计算伤害（更公平），否则由 AI 全权描述战斗过程（更具戏剧性）。
4.  **状态管理**：注意你的 HP（生命值）和 MP/能量。使用背包中的药水可以恢复状态。

## 🛠️ 技术栈

*   **HTML5 / CSS3**
*   **React 18** (通过 CDN 引入)
*   **Tailwind CSS** (通过 CDN 引入)
*   **Babel** (浏览器端 JSX 编译)

## ⚠️ 注意事项

*   **数据安全**：API Key 仅存储在你的浏览器本地缓存（LocalStorage）中，不会上传到任何服务器。
*   **网络连接**：请确保你的网络环境可以访问所选的 AI API 服务（部分服务可能需要代理）。
